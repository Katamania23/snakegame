project snake game on arduino

wat nodig: een arduino nan, ledmatrix 8x8, potentiometer, joystick, kabels.

Arduino NANO: Er zijn veel varianten van het Arduino platform, maar wij hebben een van de simpelste nodig- een chip op een printplaat met pootjes voor montage in een speciaal bord.

Ledmatrix : Onze slang gaat daarop werken. Natuurlijk kun je LCD-matrixen gebruiken om een beter beeld te krijgen, maar die zijn duurder en moeilijker te programmeren.

Potentiometer : een gewone variabele weerstand van 10oOhm. De essentie van deze apparaten is dat we de weerstand van de weerstand kunnen veranderen door aan de regelaar te draaien. In mijn project is deze weerstand verantwoordelijk voor de snelheid waarmee de slang beweegt. Als hij te snel beweegt draai ik aan de regelaar en de slang zal langzamer lopen.

Joystick : hij werkt gewoon als op ieder console. We gebruiken hem om de slang te sturen naar welke richting moet hij gaan

Waarom heb ik dit project gekozen?

Ik heb dit project gekozen omdat ik oorspronkelijk iets totaal anders van plan was. Echter, het materiaal dat ik daarvoor had besteld, zou pas midden juni geleverd worden, en dit merkte ik helaas te laat op. Daarom besloot ik aan een ander project te werken. Ik vond een YouTube-video waarin iemand mijn favoriete spel uit mijn jeugd maakte, en besloot dat na te maken. Ik volgde zijn uitleg voor de code en kopieerde een deel ervan omdat het erg moeilijk was om zelf te maken. Ik heb ook een reactie achtergelaten onder de video. Nu moet ik wachten op het bestelde materiaal dat op 30-05-2024 komt. Mijn excuses voor het vreemde excuus.
